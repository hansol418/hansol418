# 💼 주요 프로젝트 (Selected Projects)

## 1️⃣ **NotiGuard – 사내 공지 누락 문제 해결 AI 시스템**
> 실제 기업 내부 문제를 정의하고, 팝업 알림과 RAG 기반 챗봇으로 해결한 AI 서비스 프로젝트

- **설명**  
  효성전기 사내에서 반복적으로 발생하던 공지 누락과 담당자 문의 과부하 문제를 해결하기 위해  
  중요 공지를 강제 인지시키는 스마트 타겟팅 팝업과, 공지를 질문할 수 있는 RAG 기반 챗봇을 결합한  
  공지 관리 시스템을 기획 및 개발

- **핵심 기술**  
  Python, RAG, POTENS.API, Streamlit, SQLite/PostgreSQL, Railway

- **내 역할**
  - 현업 멘토 인터뷰 기반 임직원 설문 및 심층 인터뷰를 통한 문제 정의
  - 관리자용 중요공지 팝업 자동 발송 로직 설계
  - 직원용 Polling 기반 중요공지 팝업 자동 수신 로직 설계
  - 사내 공지 문서 기반 RAG 챗봇 파이프라인 구현
  - 챗봇 응답 분류 및 풀력 후처리 구현
  - 미해결 질의 자동 전달 파이프라인 설계
  - 정보화팀장 및 현업 관리자 대상 사용자 테스트 및 피드백 반영

## 2️⃣ **AI RAG Chatbot (Ko LLaMA2 + LoRA)**

- **설명**  
  사내용 문서 및 외부 검색을 기반으로 실시간 답변을 생성하기 위한  
  한국어 LLaMA2 모델 파인튜닝 및 RAG 챗봇 시스템 구축 프로젝트

- **핵심 기술**  
  Python, PyTorch, HuggingFace, LoRA(QLoRA), A100, LangGraph, Gradio

- **내 역할**
  - LangGraph 기반 StateGraph 설계
  - ReAct 방식을 30B 모델에 맞게 커스터마이징
  - ReAct Planner 추론 구조 분리
  - Serper API 기반 웹 검색 도구 직접 구현
  - 내부 문서 검색 후 30B 모델로 요약하는 RAG 파이프라인 구현
  - LangGraph 에이전트를 UI와 직접 연결하여 데모 형태로 완성
